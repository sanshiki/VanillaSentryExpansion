# VanillaSentryExpansion — 中文介绍

**VanillaSentryExpansion** 是一款 Terraria tModLoader 模组，围绕哨兵召唤师玩法扩展：更多哨兵、令旗武器、原版哨兵改动以及配件与盔甲等。

---

## 更多哨兵

新增多种哨兵法杖与对应哨兵，完善了哨兵召唤师的全游戏流程体验。

## 令旗武器

添加了对标原版鞭子的召唤师主手武器：令旗。令旗可以像鞭子一样通过左键挥击标记敌人，还可以通过右键蓄力以及插旗来召回哨兵。

## 原版哨兵改动

对原版哨兵进行不同程度的增强，在保留原版风格的前提下调整数值或行为。

## 配件与盔甲

- **配件**：哨兵锚、披肩等。
- **盔甲**：主要针对后期部分盔甲进行了哨兵召唤师分支的优化。

## 其它内容

- **召唤疲劳**：当玩家试图同时召唤多个哨兵和仆从时予以伤害减少的减益，以防过于超模。
- **多语言**：模组内支持英文（en-US）与简体中文（zh-Hans）。
- **多人适配**：模组支持多人模式游玩。
- **灾厄适配**：与灾厄模组进行了初步兼容。
